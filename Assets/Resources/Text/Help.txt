Distance Screening

* Swipe to direction optotype E to move to next level.
* Ensure you swipe before timeout progress bar triggers.
* Result is displayed on swipe mismatch optotype direction or on completion of all levels.

Color Screening

* Goal is find the color closest to either red, green, or blue.
* Select red button if vehicle color is close to red.
* Select blue button if vehicle color is close to blue.
* Select green button if vehicle color is close to green.
* Vehicle color fade away as you pass the test levels.
* Watch out for timeout progress bar, complete each level before it triggers.
* After completion of all levels, the test result is displayed.
